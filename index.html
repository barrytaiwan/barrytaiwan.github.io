Connect the blocks app


This was one of the first applications where I learned how to create my own game from online tutorials using Unity3D. From the main menu of the app, the player can select between 新遊戲 (new game)and 離開遊戲 (leave game). When the player selects 新遊戲 (new game), different colored blocks will appear. Each individual block can be selected and only one block can be selected at any given time. After pressing the block, the block can be moved to any blank space. The objective of this game is to connect as many blocks as possible to get the highest score without filling up the entire screen. Three blocks is the minimum amount of blocks that can be connected to get points. If the player does not connect the block to two other blocks and gets no points, additional random colored blocks will be added to the screen, making the game more fun to play.
